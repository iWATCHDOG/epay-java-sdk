# epay-java-sdk

## 1. 简介

epay-java-sdk是epay支付平台的Java版本SDK，提供了易派支付平台的接口调用能力，方便Java开发者调用易派支付平台的API。

## 2. sdk导入

### 2.1 maven导入

```xml

<dependency>
    <groupId>com.epay</groupId>
    <artifactId>epay-java-sdk</artifactId>
    <version>1.0.0</version>
</dependency>
```

### 2.2 gradle导入

```groovy
compile 'com.epay:epay-java-sdk:1.0.0'
```
